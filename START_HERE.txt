┌──────────────────────────────────────────────────────────────┐
│  OPTION 1 : AUTOMATIQUE (recommandé)                         │
└──────────────────────────────────────────────────────────────┘

  1️⃣  Exécutez :
      
      ./setup-dockerhub.sh

  2️⃣  Créez un token sur Docker Hub :
      https://hub.docker.com/settings/security

  3️⃣  Configurez GitLab Variables :
      Settings → CI/CD → Variables
      • DOCKERHUB_USERNAME = votre-username
      • DOCKERHUB_TOKEN = votre-token

  4️⃣  Push et c'est parti !
      
      git push origin main

┌──────────────────────────────────────────────────────────────┐
│  OPTION 2 : MANUEL                                           │
└──────────────────────────────────────────────────────────────┘

  1️⃣  Éditez .gitlab-ci.yml ligne 6 :
      IMAGE_REGISTRY: "docker.io/VOTRE_USERNAME"
                                    ^^^^^^^^^^^^
                              (remplacez ceci)

  2️⃣  Suivez les étapes 2, 3, 4 ci-dessus
